Color theory and exercises

HSL is an alternative representation of the RGB color model and it was created to align better with the way humans perceive colors. Let's take the color red for example: . Red is [255, 0, 0] in RBG space. If you wanted to change it to light red it would be hard to figure out how to change it in RGB. However, with hsl, you just have to increase the lightness value from 50% to 75%.

So what do the hue, saturation and lightness values mean?

Hue maps the colors on a scale from 0 to 360 based on the color wheel. So for example, if we want to use orange, the hue value would be 30.

Saturation shows how much the color differs from neutral gray in the color space. It ranges from 0% for fully desaturated to 100% for a pure or bright color. When we decrease the saturation, we add the amount of gray in the color. This is also called toning the color.

Lightness indicates the level of illumination. It ranges from 0% for black (no light) to 100% for full illumination that washes out the hue, so it appears white. Shading is the process of reducing the level of light below 50% (basically adding black or darkening the color). Increasing the level of light above 50% is called tinting (and it basically adds white to a color to lighten it up).
